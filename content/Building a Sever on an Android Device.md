# Building a Sever on an Android Device

###### A workshop concept by Creative Coding Utrecht held at PROTO in Eindhoven in June 2026.

In the workshop we used our old Android Devices to setup servers. In the end we were able to host our own HTML web page to the local network. But this technique can also be used to host to the world wide web. 

It is interesting because it shows that our old devices, which we dismiss as junk, still have computing power and therefore are still useful. We need to think about how we use the resources available to us and act and think in a sustainable way. Projects like these aren't meant to change the world, but to spark a social dialogue. 

The format of that workshop was surprisingly fitting to this CrossDocu format. The presentation was a projected web page with a sidebar that guided us trough the content. In addition to that they printed the whole web page as handouts. This whole setting worked really well and shows the advantages of such cross-publishing formats.
