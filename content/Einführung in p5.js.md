## Was ist p5.js?

- Nachgänger von Processing (Java)

- Weiterhin von der Processing Foundation

- Jetzt über JavaScript

- Web-basiert, im Browser

- Dadurch wesentlich zugänglicher

- Kompatibel mit anderen JavaScript Bibliotheken (d3.js, ml5.js)

- p5.js-Editor = Die eigene web-basierte Entwicklungsumgebung

- Für größere Projekte unbedingt in VS-Code o.ä. arbeiten
