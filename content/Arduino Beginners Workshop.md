# Participated at an Arduino Beginners Workshop

Hosted by the [Gründungszentrum](https://gruendungszentrum.fh-aachen.de/) FH Aachen, Summer 2026.

On two separate days we dived into some basic Arduino workflows. On the first day there was some introduction into some basics of electronics. On the second day we connected a temperature and humidity sensor and send its data onto a small display. This worked out pretty well for every participant. 

I want to mention that I really appreciated the handout they offered. We didn't use it that much while the workshop, but it is nice to have a documentation. If I ever want to use a humidity sensor I can come back to this handout. Thanks for that!
