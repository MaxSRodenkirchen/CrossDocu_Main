# Why stick with Obsidian as an editor for CrossDocu?

**Con:** 
- too many possibilities, some functions may not run in CrossDocu
- people might build systems, that do not work in CrossDocu
- one step more to publish your content
- there's no deep styling - content looks different

**Pro:**
- local files
- people only need to have one system
- no server needed
- a lot of functionality for their own system
