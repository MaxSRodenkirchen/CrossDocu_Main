 Tools for Creative Coding

Creative Coding allows us to express complex ideas visually. Two popular tools are p5.js and TouchDesigner.

 p5.js

p5.js is a JavaScript library. It makes coding accessible for artists and designers. It is great for web-based sketches -> it runs directly in the browser!

 TouchDesigner

TouchDesigner is a node-based visual programming environment. It is often used for:
- Projection Mapping
- Interactive Installations
- Real-time Audio Visuals
